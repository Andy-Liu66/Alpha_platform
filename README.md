# Alpha_platform
Create the alpha strategy and backtest

1. 目的: 利用台灣上市公司的股票創造一個投資組合，
         其中各個股票的比例藉著目前所能得到的開高低收量來計算，
         計算過程中會經過中性化的調節權重，
         以使投組達到中性化的平衡並盡力避開市場風險，
         期待可以從市場獲得超額報酬。
         
2. 功能: 程式碼中有撰寫投組策略的區域，
         可以觀察不一樣的權重比例計算帶來的獲利能力。
         也可以輕易的新增股票標的，
         以降低商品間的相關係數並達成多元化之目的。
         
3. 未來優化方向: 
         (1) 加入交易成本的計算
         (2) 建立策略回測報表
         (3) 新增股票基本面資料
